Задача 3 - «Светофор»

Создать страницу с круглым элементом, при клике на который цвет меняется в следующем порядке: зеленый - желтый - красный - зеленый - желтый - красный и т.д. Один клик — одна смена цвета. Изначально цвет элемента черный.

Дополнительно:

Решение задачи «Светофор» модифицировать таким образом, чтобы было три разных круга (как на светофоре), смена цвета происходит при клике на любой из них. Каждый круг может принимать только один из цветов. При этом, когда один из кругов становится «цветным», остальные становятся черными. Порядок смены цвета тот же, что и в изначальном задании.
